# Changelog

Todas as mudancas relevantes deste projeto serao documentadas neste arquivo.

O formato segue uma adaptacao de [Keep a Changelog](https://keepachangelog.com/pt-BR/1.1.0/) e versionamento semantico quando aplicavel.

## [0.1.2] - 2026-03-03

### Added
- Estrutura modular do firmware em multiplos arquivos `.c/.h`.
- Documentacao completa para publicacao no GitHub.
- Guia de arquitetura, deploy e protocolo.
- Templates para issues e pull requests.

### Changed
- `app_main` simplificado para foco em orquestracao.
- Build component ajustado para compilar todos os modulos.

### Notes
- Comportamento funcional foi preservado durante a reorganizacao de codigo.
